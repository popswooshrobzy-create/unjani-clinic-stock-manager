# Unjani Clinic Stock Management System: Staff Training

## 1. Welcome to the Enhanced System

### The New & Improved Stock Management System
This training covers the essential functions of the Unjani Clinic Stock Management System, focusing on the powerful new features designed to make your work easier and more efficient.

**Key Enhancements:**
- **Predictive Analytics:** Forecast stock depletion and get reorder recommendations.
- **Advanced Search & Filter:** Quickly find stock using multiple criteria.
- **Comprehensive Reporting:** Export data to CSV and PDF for auditing.
- **Enhanced User Management:** Clear roles and user deletion for authorized staff.
- **System Presentation:** Instant access to system documentation.

## 2. User Guide: Logging In & Navigation

### Accessing the System
1. **Login:** Use your existing credentials to log in.
2. **Select Dispensary:** Upon logging in, you will be prompted to select either the **Main Clinic Dispensary** or the **POD Mobile Clinic Dispensary**. This determines the stock you will manage.
3. **Navigation:** Use the sidebar to access different sections:
    - **Stock List:** View all inventory.
    - **Add Stock:** Record new stock received.
    - **Reports:** Generate audit and status reports.
    - **User Management:** Manage user access (for authorized roles).

## 3. Feature Deep Dive: Search & Filter

### Finding Stock Items Instantly
The Stock List page now features a powerful search and filter bar.

**Search Bar:**
- Type the **Item Name** or **Batch Number** to instantly narrow down the list.

**Filter Options:**
- **Category:** Filter by medication type (e.g., Analgesics, Antibiotics).
- **Expiration:** Filter by status: Valid, Expiring Soon (within 3 months), or Expired.
- **Form Type:** Filter by physical form: Syrups/Suspensions (liquids) or Tablets/Capsules (solids).

**Tip:** You can combine all filters and the search bar to find exactly what you need.

## 4. Feature Deep Dive: Predictive Analytics

### Forecasting and Reordering
The system now uses historical data to predict future needs, helping you prevent stockouts.

**Key Indicators on Stock List:**
- **Days Until Depletion:** The estimated number of days before the stock quantity reaches zero.
- **Reorder Point:** The quantity level at which you should place a new order.
- **Reorder Recommendation:** If the current quantity is at or below the Reorder Point, an **orange warning** will appear, recommending the optimal quantity to order.

**Action:** Use the reorder recommendation to place orders efficiently and maintain optimal stock levels.

## 5. Feature Deep Dive: Reports & Export

### Generating Audit and Status Reports
The Reports page allows you to generate four critical reports for auditing and planning.

**Available Reports:**
1. **Stock Inventory Report:** Full list of all items and quantities.
2. **Transaction History:** Audit trail of all stock movements (issued, received, lost).
3. **Low Stock Report:** Items below their set threshold.
4. **Expiration Report:** Items expiring soon or already expired.

**Export Options:**
- **Export CSV:** Downloads a spreadsheet file (for Excel/Google Sheets) for data analysis.
- **Export PDF:** Opens a print-ready document in a new window for easy printing or saving as a PDF document.

## 6. Feature Deep Dive: User Management

### Managing Access and Roles (Admin, Founder, Manager Only)
The User Management page provides a clear overview of the team structure.

**Key Personnel Overview:**
- Easily identify the **Clinic Founder** and **Clinic Manager**.
- See the number of **Stock Controllers** assigned to the Main Clinic and Mobile Pod.

**User Deletion:**
- **Authorized Roles** (Admin, Founder, Manager) can use the **Trash Icon** next to a user's name to remove their access.
- **Important:** Always confirm before deleting a user, as this action is permanent.
